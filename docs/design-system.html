3. Badge System

Four badge types with clear color assignments. An article shows 2-4 badges.

Type Values Color Purpose
Method Empirique, Synthèse, Théorique, Expérience vécue Blue (solid) The what — type of evidence
Voice Académique, Praticien, Organisation, Individuel Stone (light) The who — source type
Peer-Reviewed Boolean (show if true) Teal (light) Trust signal — credibility marker
Open Access Boolean (show if true) Teal (light) Trust signal — availability marker

Visual Hierarchy

  1. 1. Teal badges pop — "Peer-reviewed" and "Open Access" are trust signals for clinicians. They get the accent color.
  2. 2. Blue badges inform — Method tells you what kind of evidence. It's the primary classification.
  3. 3. Stone badges recede — Voice is useful context but not critical for first scan.

4. Category Filters

Clinical topic tags for navigation. Different from classification badges.

Categories vs. Badges

  • Badges = Classification metadata (method, voice, flags). Shown on cards.
  • Categories = Clinical topic filters (Fondements, Évaluation...). Used for navigation/filtering.
Category Filter Bar

Default state:

Tous Fondements Évaluation Présentation clinique Prise en charge Comorbidités Trajectoire

With selection:

Tous Fondements Évaluation Prise en charge Comorbidités

Implementation Note

Categories use the same pill shape as badges but serve a different purpose (filtering vs. labeling). Active filters use bg-primary-500 text-white, inactive use bg-stone-100 text-stone-600.

7. Callouts & Highlights

Left-border treatment for inline content emphasis. Different from badges.

Key Insight (Teal)

This is the key takeaway from the article. Use teal for abstracts, important quotes, or summary highlights that readers should notice.

bg-accent-50 border-l-4 border-accent-300
Reference Note (Blue)

Use blue for citation information, methodology notes, or references. More informational than emphatic.

bg-primary-50 border-l-4 border-primary-500
Neutral Note (Stone)

Use stone for translator notes, editorial comments, or other secondary information that shouldn't draw attention.

bg-stone-50 border-l-4 border-stone-300

When to Use

  • Teal: Abstracts, key quotes, "the important bit"
  • Blue: Citations, methodology, reference info
  • Stone: Translator notes, minor asides
